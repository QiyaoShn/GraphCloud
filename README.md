This is an enterprise-level cloud-based image asset management system, which is divided into three core modules: public image library, private space, and team shared space. It enables image upload and search, AI image expansion, and real-time team collaboration for image editing, providing efficient digital asset management solutions for both individuals and enterprises. The ddd-refactor branch offers the code of the DDD (Domain-Driven Design) refactored version. Moreover, the RAG intelligent customer service function is about to be added.
